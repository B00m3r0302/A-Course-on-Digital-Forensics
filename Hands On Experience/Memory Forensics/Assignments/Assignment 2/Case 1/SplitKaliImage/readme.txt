Here is an image of the Kali Linux VM while the Diamorphine rootkit is running, captured with VirtualBox.

The zipped kali image file is around 154.4M. To upload them into the github, we had to split this zip file into pieces. Used th following Mac OS X command (using terminal) to split the zipped file.

zip MyArchive.zip --out SplitArchive.zip -s 20m
